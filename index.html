HTML Table Reference

Table Tags

Tag Name Description
< table > Table The wrapper elementy for all HTML elements
< thead > Table Head The set of rows defining the column headers in a table.
< tbody > Table Body The sets of rows containing actual table data
< tr > Table row The table row container
< td > Table Data Data that'll be contain in the table row
< th > Table Header will usually be the top row of the table
< tfoot > Table Footer The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td elemty should span
Rowspan Row Span Defines how many rows a td element should span
